# COVID-19 Report APP

## Description
Just another covid-19 report app.

## Template
(https://github.com/coreui/coreui-free-vue-admin-template/blob/master/LICENSE)

#### Usage

``` bash
# serve with hot reload at localhost:8080
npm run serve

# build for production with minification
npm run build

# run linter
npm run lint

# run unit tests
npm run test:unit

# run e2e tests
npm run test:e2e

```
